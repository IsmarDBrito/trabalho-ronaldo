1 O que é o teste sala limpa? R - A estratégia e as táticas de teste sala limpa são fundamentalmente diferentes das abordagens de testes convencionais. O objetivo do teste sala limpa é validar requisitos do software demonstrando que uma amostragem estática de casos de uso foi executada de forma bem-sucedida.


2 O que faz o teste de uso estático? teste do software que determinam a distribuição de probabilidade de uso para software. A especificação para cada incremento do software é analisada para determinar uma série de estímulos que fazem o software mudar seu comportamento.
